Shopinity is a modern **e-commerce platform** built with the **MERN stack** (MongoDB, Express, React, Node.js) with **voice-assistant functionality**. It allows users to browse products, add them to the cart, and complete secure payments, all while optionally interacting via voice commands.

## ðŸ”¹ Features

- **User Authentication:** Registration, login, and secure session management.
- **Product Catalog:** Browse, filter, and search products by category, subcategory, and name.
- **Voice Assistant:** Use voice commands for searching products, adding to cart, and checkout.
- **Shopping Cart:** Add, remove, and modify items in the cart.
- **Payment Integration:** Secure checkout using Razorpay.
- **Responsive Design:** Fully responsive on mobile, tablet, and desktop devices.
- **Filtering & Sorting:** Filter products by category/subcategory, search, and sort by price.
- **Related Products:** Suggest related items based on category/subcategory.
- **Admin Features:** Manage products (add/edit/delete) through backend API.
